Bootstrap

Bootstrap is a free and open-source for faster and easier web development framework. With the basic knowledge of HTML and CSS, our team members are able to modify according to the requirement. We use bootstrap on our navigation bar. It is a fixed navigation bar. This means when we scroll down the navigation bar sticks on top of the window. We also use the enhancement of a collapsible bar, which collapses when we minimise the window.

CSS

Cascading Style Sheets(CSS) is implemented in the designed of the website for a way to define, apply, and manage sets of style characteristics. On the topic page we use CSS on the IOT Benefits image so that it zooms in when we hover on it.
